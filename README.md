# Spring Boot E-Commerce API

## Description
This project is a backend REST API for an E-Commerce system built using Spring Boot and PostgreSQL. It provides APIs for product management, order processing, and inventory management.

## Tech Stack
- Java
- Spring Boot
- Spring Data JPA
- PostgreSQL
- Docker
- REST APIs

## Architecture
The project follows a layered architecture:

Controller → Service → Repository → Database

## Features
- Product management APIs
- Order processing
- Inventory management
- RESTful API design
- Dockerized application
